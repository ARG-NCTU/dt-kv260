# dt-kv260

This repo is intended to setup a Duckiebot with Xilinx KV260

## Hostname naming rule

dt-"hardware"-"vehicle"-"number"

Ex: dt-nano-bot-01, dt-rpi-boat-03, dt-kv260-anchor-07
